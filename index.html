Podcast / 系统音频
方案 A — 标签页音频（免安装）
点击顶栏「标签页」按钮 → 在弹窗中选择播放 Podcast 的浏览器标签 → 勾选「分享标签页音频」 → 点击「分享」即可。
仅支持 Chrome / Edge，Safari 不支持。
方案 B — BlackHole 捕获全系统音频
适用于任意应用（Apple Podcast、Spotify、播客 App 等）：
  1. 安装 BlackHole 2ch：brew install blackhole-2ch
  2. 打开「音频 MIDI 设置」→ 新建「聚合设备」，勾选扬声器 + BlackHole 2ch
  3. 系统偏好设置 → 声音 → 输出改为该聚合设备
  4. 回到此页面点击「系统音频」→ 在下拉选择 BlackHole 2ch → 开始录制

使用环境预设
Podcast / 系统音频：说话节奏均匀、无长停顿。缓冲 5s 避免截断，短停顿灵敏断句。适合电台、有声书、YouTube。
最大缓冲时长（硬上限） 5.0 s
3s 积极分句15s 长段完整
不管有没有停顿，缓冲超过此时长必须强制 flush。会议建议 8s，Podcast 建议 5s。
长停顿帧数（句末静音） 2 帧 ≈ 0.5s
2帧 ≈ 0.5s 超快20帧 ≈ 5s 稳定
连续静音达到此帧数才 flush。说话间隔短时适当降低（如 2-3 帧）。
短停顿帧数（逗号/短间隔） 1 帧 ≈ 0.3s
1帧 极灵敏8帧 ≈ 2s 不敏感
VAD 轻微下降（逗号停顿）触发分句，缓冲 ≥2s 时生效。
